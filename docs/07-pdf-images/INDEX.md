Command-line tooling for raster image work and PDF/SVG conversion — format changes, resizing, and moving content between image, PDF, and vector formats.

| Tool | Description |
| --- | --- |
| [ImageMagick](01-imagemagick.md) | Command-line image suite for converting, resizing, and editing raster images across dozens of formats. |
| [img2pdf](02-img2pdf.md) | Wraps images into PDF losslessly by embedding the original JPEG/PNG data without re-encoding. |
| [pdf2svg](03-pdf2svg.md) | Converts PDF pages to SVG, preserving paths and text as scalable vector geometry. |
| [pngpaste](04-pngpaste.md) | Pastes an image from the macOS clipboard straight to a file or stdout. |

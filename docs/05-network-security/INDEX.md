Command-line scanners and benchmarks for the local network, alongside the password managers and antivirus that keep credentials and downloads safe.

| Tool | Description |
| --- | --- |
| [[01-nmap\|Nmap]] | Port scanner and network mapper for discovering hosts, open ports, and services. |
| [[02-arp-scan\|arp-scan]] | Layer-2 scanner that discovers and fingerprints every device on the local network. |
| [[03-bitwarden-cli\|Bitwarden CLI]] | Command-line client for the Bitwarden vault, for reading secrets from the terminal. |
| [[04-bitwarden\|Bitwarden]] | Desktop password manager with an end-to-end encrypted vault. |
| [[05-termius\|Termius]] | Cross-platform SSH client with a managed, syncable host list. |
| [[06-clamav\|ClamAV]] | Open-source antivirus engine that powers the Transmission download quarantine script. |
| [[07-iperf3\|iperf3]] | Active measurement tool for benchmarking TCP/UDP throughput between two hosts. |

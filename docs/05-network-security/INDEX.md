Command-line scanners and benchmarks for the local network, alongside the password managers and antivirus that keep credentials and downloads safe.

| Tool | Description |
| --- | --- |
| [Nmap](01-nmap.md) | Port scanner and network mapper for discovering hosts, open ports, and services. |
| [arp-scan](02-arp-scan.md) | Layer-2 scanner that discovers and fingerprints every device on the local network. |
| [Bitwarden CLI](03-bitwarden-cli.md) | Command-line client for the Bitwarden vault, for reading secrets from the terminal. |
| [Bitwarden](04-bitwarden.md) | Desktop password manager with an end-to-end encrypted vault. |
| [Termius](05-termius.md) | Cross-platform SSH client with a managed, syncable host list. |
| [ClamAV](06-clamav.md) | Open-source antivirus engine that powers the Transmission download quarantine script. |
| [iperf3](07-iperf3.md) | Active measurement tool for benchmarking TCP/UDP throughput between two hosts. |
| [Vault → env pattern](08-vault-to-env-pattern.md) | The store → unlock → export → inherit chain for secrets, drilled with five examples. |
| [Tailscale](09-tailscale.md) | Mesh VPN for reaching your own machines remotely and connecting into a client's machine without exposing ports. |
| [Jellyfin](10-jellyfin.md) | Self-hosted media server, reached remotely over Tailscale instead of port-forwarding. |

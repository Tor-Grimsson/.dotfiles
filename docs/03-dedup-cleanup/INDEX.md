Tools for keeping storage lean: duplicate-finders (exact and fuzzy), macOS app uninstallers that remove leftover files, and a data-recovery safety net for when a cleanup goes too far.

| Tool | Description |
| --- | --- |
| [jdupes](01-jdupes.md) | Fast exact-match duplicate file finder, an enhanced fork of fdupes. |
| [rmlint](02-rmlint.md) | Finds duplicates plus other filesystem lint and emits a reviewable cleanup script. |
| [Czkawka](03-czkawka.md) | Finds exact and similar (fuzzy) duplicate files, images, music, and videos. |
| [AppCleaner](04-appcleaner.md) | Lightweight macOS app uninstaller that removes an app and its leftover files. |
| [Pearcleaner](05-pearcleaner.md) | Open-source macOS uninstaller that also cleans leftovers from already-deleted apps. |
| [Disk Drill](06-disk-drill.md) | macOS data-recovery app for restoring deleted files from disks and external media. |

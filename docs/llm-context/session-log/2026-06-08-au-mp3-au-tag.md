# 2026-06-08 — au-mp3.sh + au-tag.sh + yq (iMac)

Two new `au-` scripts: lossless→MP3 conversion and sidecar-`.md`→audio tagging. Added `yq` as a new dep (user installed). Both verified end-to-end with ffprobe.

- `bin/au-mp3.sh`: recursive WAV/AIF/AIFF → MP3 via ffmpeg+libmp3lame, `-b` CBR (128/160/192/320, default 320), parallel `xargs -P 6`, **non-destructive** (keeps the lossless source — the deliberate inverse of `au-flac.sh`, which deletes). Verified: `-b 192` produced 192 kbps mp3s, sources kept.
- `bin/au-tag.sh`: reads a folder's sidecar `.md` YAML frontmatter (artist/album/albumartist/year/genre/cover/`tracklist[]`) via `yq --front-matter=extract`, writes ID3/Vorbis tags + embedded front cover into every mp3/flac via ffmpeg `-c copy` (no re-encode, temp-then-mv). Per-track title from `tracklist[]` (sort order) else filename (`NN <title>`), `-T` to skip. Cover: `-c` → frontmatter `cover:` → auto-detect cover/folder/front/album.{jpg,png}. Verified: all tags written, mjpeg cover stream embedded, tracklist titles applied in order. `set -euo pipefail` with 3.2-safe array handling (single assembled `ff=()` arg list, `${#meta[@]}` guard, `if`-based cover-detect loop so `set -e` doesn't trip on a trailing false test).
- New dep **yq** (mikefarah v4.53.3): added to `Brewfile` right after `jq`. `bootstrap.sh` needs no edit — it runs `brew bundle`, so the Brewfile entry installs it. User installed manually this session.

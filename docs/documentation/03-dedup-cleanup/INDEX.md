Tools for keeping storage lean: duplicate-finders (exact and fuzzy), macOS app uninstallers that remove leftover files, and a data-recovery safety net for when a cleanup goes too far.

| Tool | Description |
| --- | --- |
| [[01-jdupes|jdupes]] | Fast exact-match duplicate file finder, an enhanced fork of fdupes. |
| [[02-rmlint|rmlint]] | Finds duplicates plus other filesystem lint and emits a reviewable cleanup script. |
| [[03-czkawka|Czkawka]] | Finds exact and similar (fuzzy) duplicate files, images, music, and videos. |
| [[04-appcleaner|AppCleaner]] | Lightweight macOS app uninstaller that removes an app and its leftover files. |
| [[05-pearcleaner|Pearcleaner]] | Open-source macOS uninstaller that also cleans leftovers from already-deleted apps. |
| [[06-disk-drill|Disk Drill]] | macOS data-recovery app for restoring deleted files from disks and external media. |

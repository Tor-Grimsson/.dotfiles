# awesome-tuis

A curated list of **terminal user interface** apps, frameworks, and games — kept as a browse-and-pick source.

- **Source:** https://github.com/rothgar/awesome-tuis
- **What it is:** a community-maintained index of TUIs — dashboards, file managers, editors, monitors, games, dev tools — plus the frameworks to build them (Bubble Tea, Textual, ratatui, blessed, …).
- **Why keep it:** when looking for a new terminal tool, browse this before reaching for a GUI. Several catalog tools already came from this species (yazi, lazygit, atuin, btop, kanban-tui).

## ohmyzsh macOS plugins (to evaluate)

From the [Oh My Zsh plugin list](https://github.com/ohmyzsh/ohmyzsh/wiki/Plugins) — macOS-relevant plugins to consider for `.zshrc` `plugins=()` (already enabled: `macos`, `brew`). Enable one by adding its name to the array.

| Plugin | What |
|---|---|
| `macos` | macOS utilities (was `osx`) — Quick Look, `tab`, `cdf`, `ofd`, etc. · **enabled** |
| `brew` | aliases for common `brew` commands · **enabled** |
| `macports` | MacPorts completion + aliases |
| `iterm2` | helper functions for iTerm2 |
| `xcode` | Xcode / iOS-dev utilities |
| `bbedit` · `textmate` · `textastic` | editor integrations (BBEdit / TextMate / Textastic) |
| `marked2` · `marktext` | Markdown-previewer integrations |
| `forklift` | ForkLift (FTP app) integration |
| `droplr` | upload files / shorten links from the CLI (macOS only) |

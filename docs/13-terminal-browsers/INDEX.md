Browsers that run in the terminal instead of a window: Carbonyl renders the real modern web (JS, CSS, video) straight to terminal cells via Docker/OrbStack; w3m is the instant text-mode fallback and pipe-friendly HTML renderer.

| Tool | What it is |
| --- | --- |
| [[01-carbonyl\|Carbonyl]] | Chromium fork rendering the full web in the terminal, run via Docker/OrbStack. |
| [[02-w3m\|w3m]] | Classic text-mode browser — instant, no JS, dumps rendered pages to stdout. |

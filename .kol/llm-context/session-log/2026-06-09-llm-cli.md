## Not in the Brewfile (by design)
`llm` is `uv tool install`, not brew ‚Äî same pattern as `edge-tts` (pipx). `uv` itself is already in the Brewfile. There is **no** uv/pipx tool manifest in bootstrap; convention is doc + manual install (carried open item: pipx ‚Üí uv consolidation).

## Reproduce on the MBP
```sh
uv tool install llm
llm install llm-anthropic
llm keys set anthropic            # key from the vault
llm models default claude-haiku-4.5
```
The plugin, key, and default live in llm's own config dir (`llm keys path`), not the repo ‚Äî so they don't sync; re-run these four.

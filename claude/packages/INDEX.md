# claude/packages

Skill dependencies — canon docs a skill reads, CLI tools copied to `~/.local/bin`, and templates
copied out into new repos. Grouped by the skill family each serves.

| Group | Serves |
|---|---|
| [[kol-docs/INDEX\|kol-docs/]] | `kol-docs-fm`, `kol-docs-md`, `scaffold-docs-system`, `kol-docs-overview` — the doc-spec canon |
| [[kol-cdn/INDEX\|kol-cdn/]] | `kol-bucket-b2`, `kol-bucket-r2`, `kol-cdn-overview` — the CDN skills |
| [[kol-packages/INDEX\|kol-packages/]] | Catch-all for single skills that don't need their own top-level group (currently: `algorithmic-art`) |
| [[scaffold/INDEX\|scaffold/]] | `scaffold-dev-stack(-kol)`, `scaffold-docs-system`, `scaffold-llm-context` — the "building fresh" family |

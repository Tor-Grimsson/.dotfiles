# claude-clear

The last reply missed. Too many words, buried point, padded ending. **Restate — don't add.** Same substance, sharper delivery.

## Do
1. **Lead with the answer.** First line = the conclusion. No preamble.
2. **Cut ~50%+ of the words.** If a sentence can go, it goes.
3. **Structure it:** bullets, numbers, or a sequence — whatever lands the point fastest.
4. **One idea per line.** No paragraph-walls.
5. **Bold the load-bearing words** so it's skimmable.

## Don't
- No trailing "want me to…", "let me know…", or offers the user didn't ask for. End on the answer.
- No re-explaining what you already said — they read it, it didn't land. Say it *differently*, not *again*.
- No hedging, no caveats unless they change the decision.
- No restating the question back to them.

## Shape
**[The answer, one line.]**
- point
- point
- point

Stop there. If they want more, they'll ask.
